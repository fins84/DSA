## Synopsis
DSA assignment.
Allows user to read in graphs, perform functions on the graphs, save them and allow for journey
planning between two vertices.

## Contents
README - readme file for DSA assignment
whereNow.py - main file for program to run from (allows two different styles of running, silent and interactive)
DSAGraph.py - contains graph, vertice and edge code
DSAQueue.py - Code for queues data structure, first in first out
DSAStack.py - Code for stacks data structure, first in last out
linkedList.py - Code for linked list data structure
Menu.py - User interface, main program operations and filio IO code
DSAGraphTest.py - Test code for DSAGraph.py
DSAQueueTest.py - Test code for DSAQueue.py
DSAStackTest.py - Test code for DSAStack.py
linkListTest.py - Test code for linkedList.py
menuTest.py - Test code for menu.py
unitTest.py - All test code functions file to run from



## Dependencies
Numpy - Module that allows for implementation of arrays
MatplotLib - Module for visualising data
sys - system module for using command line arguments (argv)
re - regex, for extracting/findng data in strings
networkx - visualisation module for graphs
pickle - serialisation module for classes and objects
